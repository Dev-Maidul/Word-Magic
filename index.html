1. The difference between var, let, and const
var variable is available within the entire function where it is declared. We can redeclared it in multiple times in the same scope and can reassign values to var variables. The updated version of var is let. This is a block-scope variable only can accessable it the block where we defined it and can't redeclared it again with same varible name.const is also a variable just like var and let. But the main difference is can not reassign the value and must assign the value in the declaration time.
2.The difference between map(), forEach(), and filter()
map(), forEach(), and filter() mainly Array metheod. map() is like a loop and its return a array so that we can store it and reuse it according to our usecase whereas forEach() is a loop it's only ittarate the element from array and it's does't return anything.Lastly filter() is also a array method that also return an Array based on condition that we are finding from an array.

4. How JavaScript Promises work?
When we fetch an API it returns a Promises and it includes mainly in 3 states.1.Pending – Initial state, before the operation completes. 2.Fulfilled (Resolved) – The operation was successful. 3.Rejected – The operation failed.Once a Promise is fulfilled or rejected, it cannot change its state.
5. How closures work in JavaScript?
closures in JavaScript allows function to keep maintain private variables and create encapsulation behavior.When a function return from another function it still has access to the variables of its parent function,even after the parent function has been executed.
